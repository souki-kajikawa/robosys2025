## 概要
1つ目の数字の異なるものの中から順番を区別せずに2つ目の数字の個数選ぶ場合の数を求めるプログラムです

## クローン方法
以下のコマンドをターミナルで実行します
```
$ git clone git@github.com:souki-kajikawa/robosys2025.git
```

## 実行方法
2つの整数を入力し、1つ目の整数を異なるものの個数とし、2つ目の整数を選ぶ個数とした時の組み合わせの個数を出力します。
- 実行例
```
$ echo 5 3 | ./combination
```
- 実行結果
```
10
```
