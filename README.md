## 概要
数学の組合せ(コンビネーション)を求めるプログラムです　　

## 使用準備
以下のコマンドをターミナルで実行します
```
$ git clone git@github.com:souki-kajikawa/robosys2025.git
$ cd robosys2025
```

## 実行方法
2つの整数を入力し, 1つ目に入力された値をmとし, 2つ目に入力された値をnとした時の, 組み合わせ"mCn"を求めます.
- 実行例
```
$ echo 5 3 | ./combination
10
```
